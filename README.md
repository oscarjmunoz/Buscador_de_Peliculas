# Buscador de Peliculas
<h2>Buscador de películas funcional que utiliza la API de OMDB para obtener información detallada sobre diversas películas.</h2><br>
<h3>Herramientas utilizadas en este proyecto</h3>

Google Fonts <br>
Boxicons <br>
OMDB API: http://www.omdbapi.com

interfaz de usuario con HTML y CSS para lograr un diseño responsive, simple y fácil de usar.

JavaScript para implementar la lógica de búsqueda y presentar los resultados de manera dinámica en la página.

Solicitudes a la API de OMDB, para procesar los datos devueltos y cómo mostrar información relevante, incluyendo pósters de películas y títulos.
